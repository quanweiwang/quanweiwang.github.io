让mac系统终端支持ss的代理

七月 28, 2018

Overview

近期发现github似乎被GFW给墙了，在终端向github提交博客文章一直失败，然而终端默认不支持ss代理。
本文将介绍mac系统下如何让终端支持ss代理。

解决办法

1、搭建一个ss，具体的百度、谷歌。
2、下载ShadowsocksX-NG，具体下载地址百度、谷歌。
3、运行ShadowsocksX-NG，右键点小飞机，然后有一个Copy HTTP Proxy Shell Export Line，点击之后会copy命令。

4、到终端paste，回车即可。

这样终端就可以使用ss科学上网啦。
